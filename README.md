# Todo

Personal task manager based on following requirements:

https://paper.dropbox.com/doc/Kamana-Code-Sample-KaKg1HqgIPTDlaxb5pFTa

## Getting Started

Clone the repository.

Get dependencies.

```bash
mix deps.get
```
Run migrations

```bash
$ mix ecto.setup
```

### Start the Webserver

From the base project folder, start the server.

```bash
mix phx.server
```
